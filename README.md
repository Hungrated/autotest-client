# macaca-test-env
An exact environment for web testing, with flexbility.

initial commit on Feb 24 2018

## SETUP

```bash
# install dependencies
npm i

# see demo:
# desktop chrome test
npm run desktop

# android chrome test
npm run android

# ios safari test
npm run ios
```
