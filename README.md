# macaca-test-env
An exact environment for web testing, with flexbility.

initial commit on Feb 24 2018

## SETUP

```bash
# 1 install dependencies
npm i

# 2 see demo:
# desktop chrome test
npm run desktop

# android chrome test
npm run android

# ios safari test
npm run ios

# 3 use uirecorder:
# new record
npm run rec

# replay record
npm run rep

# delete previous js file
npm run delrec

# delete previous js file & record again
npm run rerec

```
